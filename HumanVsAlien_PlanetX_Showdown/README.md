# HumanVsAlien_PlanetX_Showdown

This project includes a generated Roblox XML place file:

- `HumanVsAlien_PlanetX_Showdown/HumanVsAlien_PlanetX_Showdown.rbxlx`

## What the `tools` folder does
- `tools/generate_rbxlx.py` builds the `.rbxlx` from your Lua source files.
- It also generates a large editable map layout (Earth base, PlanetX base, planet system, ships, fuel nodes, design anchors).
- Re-run after script edits to keep Studio place content in sync.

## Regenerate `.rbxlx`
```bash
python tools/generate_rbxlx.py
```

## Included generated world content
- Baseplate + giant arena bounds
- Earth, Moon, PlanetX, sun backdrop (space scene)
- EarthBase with research, rocket pad, EMP array, walls, spawns
- PlanetXBase with death-ray core, armory, conduits, spires, spawns
- Human + Alien ship fleets
- 60 fuel nodes for objective loops
- 400 Hyperwave design nodes for manual art pass anchors
